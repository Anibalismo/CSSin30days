Día #6 Estilos de impresión

Caja de arena ¡Juega acá!

Resumén:

Usar CSS para crear un estilo de impresión. En el "print preview" del navegador la página se vería mal por los fondos de color. Un estilo de impresión puede arreglar eso.

Bitsofcode. Articles on Frontend Development

Información Más datos útiles

¿Qué estamos haciendo?

Estableciendo un formato para que al imprimir la página se vea bien :)

¿Quién se beneficia?

Si se espera que la página se imprima, el estilo de impresión mejora mucho la calidad visual del documento.

¿Qué tan compatibles es esto con otros navegadores?

Puedes verificar la compatibilidad de los estilos de CSS en este sitio web. Éste permite ver la compatibilidad de cada recurso CSS por versión de cada navegador.
